# Changelog

## 1.0.0 - 2026-06-20

- Reworked the app from a MEDSIS-focused mock data generator into a broader synthetic healthcare data generator.
- Added healthcare profiles for administration, clinical research, medical professional workflows, public health surveillance, and mixed datasets.
- Expanded rule inference using field name, field comment, data type, and allowed values.
- Added healthcare-specific mock values for diagnoses, procedures, medications, labs, vitals, payers, facilities, research fields, and public health fields.
- Added deterministic seed support.
- Added date-range controls.
- Added CSV, JSON, and XLSX export.
- Added built-in sample dictionaries.
- Added GitHub Pages-ready repository files.
